Central Server(中心服务器):
    HeartbeatService: 
        Central heartbeat service, to record which device is online.
        Every 5 second send a boardcast message. If the device is online, it should feedback a heartbeat message. If no response 3 times, the device is offline.
        
        Boardcast msg: 
            Byte0: 0x0A Boardcast head 
            Byte1: 0x01 Find
            Byte2: Unused
            Byte3: Unused
        Response msg: 
            Byte0: 0x0A Boardcast head 
            Byte1: 0x02 Response 
            Byte2: Device type. 0x01-Lamp 0x02-Camera
            Byte3: Device ID
            Byte4: 0x00 Offline, 0x01 Online
            Byte5: Len
            Byte[6:6+len]: Device Name
            ...

Raspberry to Lamp
    Discription: Raspberry send a command to control a lamp on or off.
                 The lamp respond a msg if everything is ok.
    Byte0: 0x0C Led control msg head
    Byte1: 0x01 Instant, 0x02 Crontab
        Instant
            Byte2: 0x01 On, 0x02 Off
        Crontab
            Byte2: 0x01 On, 0x02 Off
            Byte3: Time Hour
            Byte4: Time Minutes
            Byte5: 0x01 On, 0x02 Off
            Byte6: Time Hour
            Byte7: Time Minutes 
    Lamp Respond: 0x0C 0x00 Means everything is ok.
                  0x0C 0x-- Means something happend.
        
Raspberry to STB
    Discription: Raspberry send a command to STB for play a video by tcp protocol.
                 STB mount to the samba server and play the video.
    Byte0: 0x70 Play video msg head
    Byte1: Len
    Byte2-Byte2+len: Video file name
    
    STB respond: 0x70 0x00 Means everything is ok.
                 0x70 0x-- Means something happend.
